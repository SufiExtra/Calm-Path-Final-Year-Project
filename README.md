# 📖 Overview

**Calm Path** is a full-stack AI-powered mental wellness mobile application developed to help users monitor and improve their mental health through Machine Learning and online psychiatric support.

The application predicts **stress** and **depression levels** using Machine Learning models and allows users to connect with psychiatrists through:

* 📹 Video Consultations
* 🏥 Physical Clinic Visits
* 📅 Online Appointment Booking

The project focuses on combining **Artificial Intelligence**, **Mobile Application Development**, and **Mental Health Support** into a single interactive platform.
